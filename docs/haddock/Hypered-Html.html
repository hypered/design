nav :: Html -> Html Source #

The main horizontal navigation component. It uses "flex justify-between" so it is better to group its contained links with divs. With a single div, they are all grouped on the left. With two, the first is grouped to the left and the second to the right. See https://hypered.github.io/design/storybook/?path=/story/navigation--navigation for examples.
